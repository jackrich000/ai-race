# The AI Race

Up-to-date charts tracking frontier AI model performance across leading benchmarks over time.

**Live site:** [ai-race.vercel.app](https://ai-race.vercel.app/)

Tracks 5 labs (OpenAI, Anthropic, Google DeepMind, xAI, Chinese Leaders) across 10 benchmarks in three views: AI Frontier (best score across all labs), Lab Race (per-lab comparison), and Cost of Intelligence (cheapest model above a quality threshold).

## Tech stack

- Plain HTML/CSS/JS with Chart.js (no build step)
- Supabase (Postgres + row-level security) for data storage
- Vercel for hosting (auto-deploys on push)
- Node.js scripts for data ingestion and analysis generation

## Project structure

```
lib/
  config.js            Shared constants: labs, benchmarks, time helpers (used by browser + Node)
  analysis-prompt.js   LLM system prompt for generating AI commentary
data-loader.js         Browser: fetches data from Supabase, builds chart-ready structures
app.js                 Browser: Chart.js rendering, UI interactions, export canvas
styles.css             All styling (dark theme, fluid scaling, mobile breakpoints)
index.html             Single-page entry point
api/
  og.js                Vercel Edge function generating Open Graph social preview image
scripts/
  update-data.js       Data ingestion: fetches from 5 sources, computes cumulative best, writes to Supabase
  generate-analyses.js Pre-generates AI commentary (structured stats + Claude headline/commentary per mode)
  seed-manual.js       One-time seed for manually-curated benchmarks (HumanEval, SWE-bench Pro)
```

## Data pipeline

The ingestion script (`scripts/update-data.js`) fetches from 5 sources in parallel:

1. **Artificial Analysis API** - HLE, GPQA Diamond scores + pricing data
2. **SWE-bench GitHub** - SWE-bench Verified leaderboard
3. **ARC Prize** - ARC-AGI-1, ARC-AGI-2 leaderboard
4. **Epoch AI** - AIME, ARC-AGI, SWE-bench, FrontierMath, MATH Level 5 (from downloadable CSV bundle)
5. **Cost cross-reference** - Matches Epoch scores with Artificial Analysis pricing

After fetching, the script merges all sources, filters out model card duplicates where verified data exists, computes cumulative-best scores per (benchmark, lab, quarter), and writes to Supabase via delete + insert.

AI analyses are pre-generated by `scripts/generate-analyses.js`, which computes stat callouts in code and calls Claude for headline/commentary. Results are cached in Supabase; the frontend makes no live API calls.

## Running locally

Start the preview server:

```bash
node server.js
# Opens at http://localhost:3002
```

Run data ingestion (requires `.env` file with `SUPABASE_SERVICE_KEY` and `AA_API_KEY`):

```bash
node scripts/update-data.js
```

Regenerate AI analyses (requires `.env` with `SUPABASE_SERVICE_KEY` and `ANTHROPIC_API_KEY`):

```bash
node scripts/generate-analyses.js              # all presets
node scripts/generate-analyses.js all-time     # single preset
```

## Adding a new benchmark or lab

All benchmark and lab definitions live in `lib/config.js`. Adding a new benchmark is a single-file change to `BENCHMARK_META` (plus wiring up the data source in `scripts/update-data.js`). Adding a new lab is a single-file change to `LABS`.
